# Server-Killer
## 一. 使用方法
`1`. 使用clang/gcc编译(这里用clang做示范)  
```sh
clang++ main.cpp -o killer
```
`2.0`. 运行(不带参数)  
```sh
./killer
```
`2.1`. 运行(带参数)  
```sh
./killer 服务器地址 端口号 线程数量
```  
`3`. 按`Enter`键
